project ini dibuat untuk memudahkan mengakses timbangan lantai di lini produksi , rencananya timbangan digital akan dikoneksikan dg wifi dan mengirim data berat ke server.
produk timbangan yg sudah support wifi harganya mahal, jadi kita gabungkan nodemcu esp8266 dengan timbangan lantai merek sonic A12E ini agar bisa terkoneksi dengan wifi perusahaan.
komponen yang dibutuhkan :
1. ESP8266 (boleh juga ESP32)
2. RS232 TTL (Male)
3. Kabel jumper female to female 4 pcs
4. timbangan digital dengan output port DB9/RS232

Software yang digunakan :
1. Arduino IDE
2. Visual Studio Code
3. XAMPP
